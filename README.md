# Emoji Matching Game

A memory matching game with 24 themed emoji categories and progressive difficulty levels. Match pairs of emojis across a 6x7 grid, track your streaks, and advance through levels.

Originally handwritten in [CodePen](https://codepen.io/bwilf9/pen/gOGLVmO/ed4e13a857d8f6d77a1035d329172c1c), now deployed at [emoji-matching-game.bwilf.app](https://emoji-matching-game.bwilf.app).

## How to Play

- Click a tile to reveal its emoji
- Click a second tile to try to find a match
- Matched pairs fade away; mismatches reset
- Clear all 21 pairs to advance to the next level
- Each level draws from a different emoji category (faces, animals, food, sports, etc.)

## Tech

Vanilla HTML, CSS, and JavaScript. No build step, no dependencies.

## Development

Open `src/index.html` in a browser, or run locally with:

```sh
npx serve src
```
